== Description ==

= Latest Posts/CPTs Listing =

Any Post/CPT Listing allows you to add latest posts or custom posts to your website as section with gutenberg builder. Display settings allows you to select different style from list or gird view with other settings like limiting of number of posts on front. 

It is now supports the Multisite. So you can use directly on individual sites.

== Installation ==

⚠️ NOTE: Make sure you are using <a href="https://wordpress.org/plugins/gutenberg/">Gutenberg Block Editor</a> which is the current default page editor, and not Classic editor or any other Page Builder for WordPress. 

1. Upload `any-cpt-listing-gutenberg-block.zip` to the `/wp-content/plugins/` directory and extract it.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Search for <strong>'Any post listing'</strong> or you can find it under widget category, add using the ‘Add Block’ button in your page editor.

== Frequently Asked Questions ==

= <span id="faq">Is Gutenberg required to use this plugin ?</span> =

Well, Gutenberg is kind of the default page editor for WordPress version 5.0 and further. So if you haven’t messed up with default page builder settings, you don’t need any installations. However, if you have, then go and download [Gutenberg Plugin](https://wordpress.org/plugins/gutenberg/), install it and keep it activated.

= Will it work with all themes out there? =

Yes definitely! We have designed the blocks to have compatibility with all the themes out there and we have made tags and CSS structure non-conflict in a manner so that you can enjoy designing websites in your own flow regardless of worrying about the theme.

= How many blocks I can create? =

Unlimited, There is no limit of creation of blocks. You can use it anywhere in the site.  
 
= Does it required coding knowledge ? =

No coding knowledge required. It is very easy as well as simple to plug & play. 

= Can I add my own class to this block? =

Yes, you add extra class from block's addition setting.

= Can I added extra CSS for this block? =

Yes, You can override default css by using unique parent class or you can also add your custom class from addition setting. 

= Can I use this for my Multisite? =

Yes.
